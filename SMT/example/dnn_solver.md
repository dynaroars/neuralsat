# Deep Neural Network Solver

### Overview

1. Architecture
    
    **Custom** `DPLL` <===> `Linear` solver (`Simplex` (phase 1) <=> **Standard** `DPLL`)

2. Custom DPLL:
    - Start with `formula=[]`, set of unassigned variables `vars={1, 2, ...}`
    - Stop when **all variables** are assigned

3. Procedure

    - Loop:
        - Propagate (DPLL + Theory)
        - Check SAT
        - Decide

### Example

##### Input

- DNN input format
```
dnn = {
    'a00': [1, '1x0 - 1x1'],
    'a01': [2, '1x0 + 1x1'],
    'a10': [3, '0.5n00 - 0.2n01'],
    'a11': [4, '-0.5n00 + 0.1n01'],
    'y0': '1n10 - 1n11',
    'y1': '-1n10 + 1n11',
}
```
- Conditions format
```
conditions = {
    'in': '(and (x0 < 1) (x1 > 2))',
    'out': '(y0 > y1)'
}
```
